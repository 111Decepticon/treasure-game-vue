# 🏴‍☠️ 多地点寻宝游戏

Vue 3 重构版本的多地点寻宝游戏，探索不同地点，解开谜题，寻找传说中的宝藏！

## ✨ 功能特性

- 🗺️ 6个独特地点探索
- 🎯 交互式谜题解答  
- 🎵 完整的音频系统
- 💾 自动进度保存
- 📱 响应式设计

## 🎮 游戏地点

1. **古老图书馆** - 寻找初始线索
2. **失落神庙** - 解码古代文字
3. **神秘洞穴** - 避开守卫陷阱
4. **沉船海滩** - 寻找藏宝图
5. **远古山脉** - 解读星象线索
6. **宝藏密室** - 打开宝藏箱

## 🚀 在线体验

访问 [GitHub Pages](https://你的用户名.github.io/treasure-game-vue) 开始游戏！

## 🛠️ 技术栈

- **前端框架**: Vue 3 + 组合式API
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **构建工具**: Vite
- **部署平台**: GitHub Pages

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
